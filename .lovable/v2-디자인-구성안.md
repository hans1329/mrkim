## 2. 컬러 시스템

### 2.1 기본 팔레트

```css
:root[data-theme="v2"] {
  /* 순백 기반 */
  --v2-bg: 0 0% 100%;              /* #FFFFFF */
  --v2-surface: 0 0% 100%;         /* 카드도 순백 */
  --v2-text: 0 0% 13%;             /* #222222 */
  --v2-text-secondary: 0 0% 46%;   /* #757575 */
  --v2-border: 0 0% 92%;           /* #EBEBEB 미세 구분선 */
  --v2-divider: 0 0% 96%;          /* #F5F5F5 영역 구분 */
}
```

### 2.2 애플 그라데이션 팔레트

핵심 포인트 컬러. iOS 18 / macOS Sequoia 스타일의 멀티컬러 그라데이션.

```css
:root[data-theme="v2"] {
  /* 메인 그라데이션 (기상도 앵커, CTA, 보이스 버블) */
  --v2-gradient-main: linear-gradient(135deg, #007AFF 0%, #5856D6 40%, #AF52DE 70%, #FF2D55 100%);

  /* 서브 그라데이션 (카드 액센트, 호버) */
  --v2-gradient-soft: linear-gradient(135deg, #007AFF20 0%, #5856D620 40%, #AF52DE20 70%, #FF2D5520 100%);

  /* 상태별 그라데이션 */
  --v2-gradient-success: linear-gradient(135deg, #34C759 0%, #30D158 100%);
  --v2-gradient-warning: linear-gradient(135deg, #FF9500 0%, #FF6B00 100%);
  --v2-gradient-danger: linear-gradient(135deg, #FF3B30 0%, #FF2D55 100%);

  /* 개별 애플 시스템 컬러 (HSL) */
  --v2-blue: 211 100% 50%;         /* #007AFF */
  --v2-indigo: 242 56% 59%;        /* #5856D6 */
  --v2-purple: 280 62% 60%;        /* #AF52DE */
  --v2-pink: 346 100% 59%;         /* #FF2D55 */
  --v2-green: 142 69% 50%;         /* #34C759 */
  --v2-orange: 35 100% 50%;        /* #FF9500 */
  --v2-red: 4 100% 59%;            /* #FF3B30 */
}
```

### 2.3 기상도별 그라데이션 오버라이드

기상도 앵커의 배경 그라데이션만 날씨 상태에 따라 변경.

```css
/* ☀️ 맑음 — 블루→인디고 (안정) */
[data-weather="sunny"]   { --v2-weather-gradient: linear-gradient(135deg, #007AFF, #5856D6); }

/* 🌤️ 구름 조금 — 인디고→퍼플 (주의) */
[data-weather="partly-cloudy"] { --v2-weather-gradient: linear-gradient(135deg, #5856D6, #AF52DE); }

/* ⛅ 흐림 — 퍼플→핑크 (경고) */
[data-weather="cloudy"]  { --v2-weather-gradient: linear-gradient(135deg, #AF52DE, #FF2D55); }

/* 🌧️ 비 — 핑크→레드 (위험) */
[data-weather="rainy"]   { --v2-weather-gradient: linear-gradient(135deg, #FF2D55, #FF3B30); }

/* ⛈️ 폭풍 — 레드 단색 펄스 (긴급) */
[data-weather="storm"]   { --v2-weather-gradient: linear-gradient(135deg, #FF3B30, #D70015); }
```

## 3. 타이포그래피

Pretendard (한글) + SF Pro Display (영문/숫자) 조합.

```
--v2-text-hero:    72px / 1.0  / 800  → 풀스크린 매출
--v2-text-display: 48px / 1.1  / 700  → 피드 핵심 숫자
--v2-text-title:   24px / 1.3  / 700  → 카드 제목
--v2-text-body:    16px / 1.5  / 400  → 본문
--v2-text-caption: 13px / 1.4  / 400  → 보조
--v2-text-micro:   11px / 1.3  / 500  → 칩, 배지
```

숫자 표현: `127만원` → "127" = 48px bold, "만원" = 16px regular
변화율: `+23%↑` → 24px bold + 그라데이션 텍스트 (상승=green, 하락=red)

## 4. 그라데이션 적용 포인트

### 4.1 기상도 앵커 (Weather Anchor)

```
┌─────────────────────────────────┐
│  ☀️  오늘 매장 날씨: 맑음   👤   │  배경: var(--v2-weather-gradient)
│  ████████████░░░░ 절세 78%      │  게이지: 그라데이션 fill
└─────────────────────────────────┘

• 배경: var(--v2-weather-gradient), 텍스트 화이트
• 절세 게이지: var(--v2-gradient-main) fill
• 모서리: rounded-2xl
• 높이: 80px (3축 정보 포함)
```

### 4.2 피드 카드

```
┌────────────────────────────────┐
│  사장님, 어제 카드매출이         │  배경: #FFFFFF
│                                │
│    127만원                     │  Big Typography, 텍스트 #222
│                                │
│  전주보다 +23% ↑               │  ← 그라데이션 텍스트
│  ─────────────────────         │  ← 1px #EBEBEB
│  📊 상세 보기                   │  ← 탭 시 그라데이션 언더라인
└────────────────────────────────┘

• 배경: 순백
• 쉐도우: 0 1px 3px rgba(0,0,0,0.04), 0 4px 12px rgba(0,0,0,0.03)
• 호버/포커스: border 1px에 var(--v2-gradient-soft) 오버레이
• 모서리: rounded-2xl
• 카드 간 간격: 12px
```

### 4.3 보이스 버블

```
      ┌─────┐
      │ 🎙️  │  56px 원형
      └─────┘
  [영수증] [매출] [세금]

• Idle: 순백 배경 + 미세 쉐도우 + 그라데이션 보더 (1px)
• 탭 시: 배경 var(--v2-gradient-main), 아이콘 화이트
• 홀드 시: scale(1.3) + 그라데이션 글로우 + 파형
• 컨텍스트 칩: 순백 + 1px border + 탭 시 그라데이션 배경
```

### 4.4 미처리 액션 카드 (Sticky)

```
│ ⚡ 미처리 2건                    │
│ ┌──────────┐ ┌──────────┐      │
│ │부가세 D-3│ │영수증 5건│      │
│ │🔴        │ │🟡        │      │
│ └──────────┘ └──────────┘      │

• 긴급: 좌측 3px 보더 var(--v2-gradient-danger)
• 주의: 좌측 3px 보더 var(--v2-gradient-warning)
• 완료: 좌측 3px 보더 var(--v2-gradient-success)
```

## 5. 쉐도우 & 깊이

순백 배경에서 깊이감은 오직 쉐도우로.

```css
--v2-shadow-sm: 0 1px 2px rgba(0,0,0,0.03);          /* 칩, 작은 요소 */
--v2-shadow-md: 0 1px 3px rgba(0,0,0,0.04),
                0 4px 12px rgba(0,0,0,0.03);           /* 카드 */
--v2-shadow-lg: 0 4px 6px rgba(0,0,0,0.04),
                0 12px 32px rgba(0,0,0,0.06);          /* 오버레이, 보이스 */
--v2-shadow-glow: 0 0 20px rgba(0,122,255,0.15);      /* 그라데이션 글로우 */
```

## 6. 모션 & 애니메이션

| 요소 | 트리거 | 모션 | 듀레이션 |
|------|--------|------|----------|
| 피드 카드 | 스크롤 진입 | translateY(20) → 0 + fade | 400ms spring |
| 기상도 변경 | 데이터 갱신 | 그라데이션 컬러 전환 | 800ms ease |
| 절세 게이지 | 미션 완료 | 그라데이션 fill + 카운트업 | 600ms |
| 보이스 활성 | 홀드 | scale(1→1.3) + 글로우 확산 | 200ms spring |
| 축하 이벤트 | 최고 매출 등 | 컨페티 파티클 | 2000ms |
| 딥워크 진입 | 칩/음성 | translateY(100%) → 0 | 500ms spring |

prefers-reduced-motion 시 모든 모션 최소화.

## 8. V1과의 공존

V2 토큰은 `[data-theme="v2"]` 스코프 안에서만 적용. V1 무영향.

```css
[data-theme="v2"] {
  --background: 0 0% 100%;
  --foreground: 0 0% 13%;
  /* 나머지 V2 토큰... */
}
```

`/v2` 라우트 진입 시 `<div data-theme="v2" data-weather="sunny">` 래퍼 적용.
