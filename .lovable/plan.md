

# 김비서 (Kim Secretary) AI ERP 앱 개발 계획

> "사장님은 명령만 하세요. 실행은 김비서가 합니다"

소상공인을 위한 AI 기반 백오피스 자동화 앱으로, 대화형 인터페이스와 대시보드를 결합한 하이브리드 디자인으로 구현합니다.

---

## 🎯 핵심 기능 개요

### 1. 대화형 AI 에이전트 (김비서)
- **자연어 명령 처리**: "직원 김OO 퇴사 처리해줘" → 자동으로 필요한 단계 안내 및 처리
- **실시간 경영 브리핑**: "오늘 매출 현황", "이상 결제 있어?" 등 즉시 응답
- **업무 가이드**: 복잡한 행정 절차를 단계별로 안내

### 2. 프로그래머블 머니 (자금 자동화)
- **세금 자동 분리**: 매출 발생 시 부가세(10%) 자동 계산 및 예치금 관리
- **조건부 자동 이체**: 급여일, 납품 완료 등 조건 충족 시 자동 실행
- **자금 현황 추적**: 운영자금, 예치금, 예정 지출 실시간 모니터링

---

## 📱 화면 구성

### 메인 대시보드
- **오늘의 요약 카드**: 매출, 지출, 순이익 한눈에
- **예치금 현황**: 부가세, 급여 적립금 등 목적별 자금 현황
- **예정된 자동이체**: 다가오는 급여일, 결제일 리스트
- **최근 알림**: 이상 결제, 중요 업무 알림

### AI 채팅 인터페이스 (하단 또는 사이드)
- **플로팅 채팅 버튼**: 언제든 김비서와 대화
- **음성 입력 지원** (선택사항)
- **대화 히스토리**: 이전 명령 및 처리 결과 조회

### 세부 관리 페이지
- **매출/매입 관리**: 거래 내역, 세금계산서 관리
- **직원 관리**: 급여, 4대보험, 입퇴사 처리
- **자금 규칙 설정**: 자동 이체 조건 설정

## 💡 김비서 대화 예시

| 사용자 명령 | 김비서 응답 |
|------------|-----------|
| "오늘 매출 얼마야?" | "오늘 총 매출은 1,250,000원입니다. 카드 85%, 현금 15%입니다." |
| "직원 김민수 퇴사 처리해줘" | "퇴사 처리를 도와드릴게요. 퇴사일을 알려주세요. 이후 4대보험 상실신고와 퇴직금 계산을 안내해드립니다." |
| "이번달 부가세 얼마나 쌓였어?" | "이번 달 부가세 예치금은 2,340,000원입니다. 납부 예정일은 1월 25일입니다." |
| "거래처 A상사에 납품 완료되면 결제해줘" | "납품 완료 조건부 이체를 설정했습니다. A상사, 5,000,000원, 납품 확인 후 즉시 이체됩니다." |

## 🔧 기술 구현 (시뮬레이션 모드)

### 데이터 구조
- **거래 내역**: 매출/매입 시뮬레이션 데이터
- **직원 정보**: 급여, 입사일, 4대보험 정보
- **자금 규칙**: 자동 이체 조건 및 예치금 설정
- **대화 히스토리**: AI 대화 기록

### AI 기능
- Lovable AI를 활용한 자연어 처리
- 문맥을 이해하는 멀티턴 대화
- 데이터 기반 경영 인사이트 제공

## 🎨 디자인 컨셉

- **하이브리드 레이아웃**: 좌측 사이드바 + 중앙 대시보드 + 우측/하단 AI 채팅
- **친근한 비서 톤**: 전문적이면서도 따뜻한 대화체
- **직관적인 시각화**: 차트와 카드로 한눈에 파악
- **모바일 반응형**: 사장님이 이동 중에도 사용 가능

## 📋 개발 순서

1. **기본 레이아웃**: 사이드바, 대시보드, AI 채팅창 구조
2. **대시보드**: 매출/지출 요약, 예치금 현황, 알림
3. **AI 채팅 기능**: 자연어 명령 처리 및 응답
4. **프로그래머블 머니**: 세금 자동 분리, 조건부 이체 시뮬레이션
5. **세부 관리 페이지**: 매출/매입, 직원, 자금 규칙 관리
