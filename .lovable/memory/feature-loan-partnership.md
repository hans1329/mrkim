## 대출 연동 전략

### 연동 방식: 제휴 리드 전달 (Affiliate Lead Generation)
- 직접 대출 중개 ❌ → 대부중개업 라이선스 불필요
- 광고/마케팅 제휴 형태로 계약
- 수수료 명목: 중개 수수료 ❌ → 광고비/마케팅비 ⭕

### 제휴 대상 (MVP)
- **핀테크 대출 비교**: 핀다, 토스, 뱅크샐러드 등
- **정책자금 안내**: 소상공인진흥공단, 신용보증기금 (외부 링크)

### 데이터 흐름
1. 코드에프로 신용정보 조회 → 대출 가능 여부 사전 체크 (선택)
2. 고객 동의 하에 기본 정보만 전달 (이름, 연락처, 사업자번호)
3. 민감 금융정보는 수집/전달 ❌
4. 실제 대출 심사/실행은 제휴사에서 100% 처리

### 법적 고지 (UI 필수 표시)
- "김비서는 대출 중개업자가 아니며, 제휴사 연결 서비스만 제공합니다"
- "대출 심사 및 실행은 해당 금융사/핀테크사의 책임 하에 진행됩니다"
- 개인정보 제3자 제공 동의 필수

### 수익 모델
- 제휴사로부터 리드당 광고비 수취
- 대출 실행 시 성과 기반 마케팅비 (CPA)
