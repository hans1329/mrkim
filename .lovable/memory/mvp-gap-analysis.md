## 구현 우선순위 제안

### Phase 1: 인증 + 데이터 연동 기반
1. 회원가입/로그인 (Cloud Auth)
2. 커넥터 플랫폼 공통 구조
3. 코드에프 연동 (홈택스/은행/카드)

### Phase 2: 대시보드 고도화
1. 데이터 신뢰 표시 (출처/갱신시각)
2. 연동 부족 안내 UX
3. 온보딩 연동 타입 확장

### Phase 3: AI Chat 고도화
1. 질문 카탈로그 + 라우팅
2. 답변 근거 표시
3. 질문/응답 로그

### Phase 4: 모바일 + 운영
1. 푸시 알림 구현
2. Braze CRM 연동
3. 관리자 콘솔
