정기 세무 신고 워크플로우는 2025/2026년 부가가치세 확정신고 요건을 기반으로 설계된 체크리스트와 일정 관리 기능을 제공함.

## 기본 요청자료 (6항목)
개인/법인 구분하여 각각 6개 항목 표시:
1. 종이세금계산서 (invoice_ready)
2. 카드매출/현금매출 (card_ready)
3. 신용카드 매입자료/현금영수증 (purchase_ready)
4. 고정자산 100만원 초과 (fixed_asset_ready)
5. 비용 지출 영수증/송금 명세서 (expense_ready)
6. 통장 거래내역 (bank_ready)

## 업종별 추가 요청자료
business_type 키워드 매칭으로 동적 표시:
- 수출/수입/무역/도소매 → 수출·수입 서류 2항목
- 부동산/임대 → 임대차계약서, 월세 세금계산서
- 건설/시공/인테리어 → 진행중·신규 공사건 계약서
- 전자상거래/온라인/쇼핑몰 → 플랫폼별 매출 내역
