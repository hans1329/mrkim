# Memory: feature/ai-outbound-call
Updated: 2026-01-31

## AI 대리 통화 기능 (단방향)

### 개요
김비서가 사장님을 대신해 고객/거래처에 전화를 걸어 안내 메시지를 전달하는 기능.
양방향 대화가 아닌 **단방향 TTS 전달** 방식으로 자연스러움 유지.

### 기술 스택
- **발신**: Twilio Voice API
- **음성 합성**: ElevenLabs TTS (eleven_multilingual_v2)
- **로그 저장**: Lovable Cloud DB

### 통화 흐름
```
1. 사용자 요청 또는 스케줄 트리거
   ↓
2. Edge Function: 메시지 생성
   ↓
3. ElevenLabs TTS: 음성 파일 생성
   ↓
4. Twilio: 발신 + 음성 재생
   ↓
5. 통화 종료 → 결과 DB 저장
   ↓
6. 알림 센터에 결과 표시
```

### 통화 유형
| 유형 | 트리거 | 메시지 예시 |
|------|--------|-------------|
| 예약 확인 | 수동/스케줄 | "안녕하세요, OO가게입니다. 내일 오후 3시 예약 확인차 연락드렸습니다." |
| 미수금 안내 | 수동/스케줄 | "안녕하세요, 지난 OO 대금 OO원이 미수 상태입니다. 확인 부탁드립니다." |
| 배송 안내 | 자동 | "안녕하세요, 주문하신 상품이 발송되었습니다." |
| 리마인더 | 스케줄 | "내일 정기 방문 예정입니다. 확인 부탁드립니다." |

### 통화 상태
- `pending`: 통화 대기
- `calling`: 통화 시도 중
- `connected`: 연결됨 (재생 중)
- `completed`: 통화 완료
- `no_answer`: 부재중
- `busy`: 통화중
- `failed`: 실패

### 모니터링 UI
- 통화 목록 (누구에게, 언제, 왜)
- 전달 내용 (TTS 스크립트)
- 통화 상태 및 시간
- 알림 센터 연동 (완료/실패 알림)

### 법적 고려사항
- 발신자 번호: Twilio 등록 번호 또는 사업장 번호
- 통화 녹음 시: 상대방 동의 필요 (선택 기능)
- 스팸 방지: 일일 발신 제한, 야간 발신 제한
